# Task Planner Application

Task Planner is a simple desktop application developed in Python using Tkinter for managing daily tasks. It allows users to add, edit, delete, and view tasks with scheduled due dates and reminders.

## Features

- **Create Tasks:** Users can add tasks with a description, due date, and additional notes.
- **Edit Tasks:** Tasks can be modified after being created.
- **Delete Tasks:** Users can remove tasks no longer needed.
- **View Tasks:** All tasks are viewable in a sidebar, arranged by their due dates.
- **Reminders:** The application provides reminders when tasks are due.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. 

### Prerequisites

You need Python installed on your system. The application is developed with Python 3.11. Any Python 3.x version should be compatible. Ensure Tkinter is available on your Python installation (usually included by default).

## Usage

Once the application is running:

1. **Add a Task:** Click on the "Planner" button, then enter the task details including the date and time.
2. **View Tasks:** View all tasks in the sidebar of the main application window.
3. **Edit or Delete Tasks:** Click on a task in the sidebar and select the desired option.
